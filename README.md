# mlflow-test-project

## How to run

1. Ensure you have installed `mlflow` library `pip install mlflow` on your system / virtual environment
2. Open a terminal and run the following command `mlflow run [GITHUB_LINK] --experiment-name [EXP_NAME]`
   1. For example : `mlflow run https://github.com/maxlee98/mlflow-test-project --experiment-name Loan_prediction`
